Detailed Description

template<class Heuristic, class NodeT>
class Search< Heuristic, NodeT >

A concrete search interface, for performing a tree search using a heuristic to evaluate each individual position, which manages the nodes of the search tree.

Template Parameters
HeuristicThe class evaluating each position.
NodeTThe class representing a single node in the search tree.
