Detailed Description

template<std::size_t N>
class NInARow::VectorizedBitsetCounter< N >

Counts the number of overlapping bits between a given bitset and a vector of known bitsets in an efficient, vectorized way. Uses size_t instead of actual bits to keep track of total overlap counts in the final evaluation.

Template Parameters
NThe maximum length of all of the bitsets in the known vector of bitsets.
