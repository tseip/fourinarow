Detailed Description

template<std::size_t HEIGHT, std::size_t WIDTH, std::size_t N>
class NInARow::Board< HEIGHT, WIDTH, N >

Represents a board of N-in-a-row. In a game of N in a row, players alternate placing stones on a rectangular board. When a player places N stones in a row, either orthogonally or diagonally, that player wins.

Template Parameters
HEIGHTThe height of the board.
WIDTHThe width of the board.
NThe number of stones in a row required to win.
