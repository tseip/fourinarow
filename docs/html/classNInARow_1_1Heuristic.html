A helper function for allowing the heuristic to apply its lapse rate to a given move selection. Does not perform any evaluation itself - assumes that the given tree has already been built up by a search and returns either the best move in the tree, or a lapsed random move if the lapse rate determines we should.

Parameters
treeThe tree of moves over which we'd like to select the best.
Returns
Either the best move in the given tree, or a random move if we lapse.
